# Piscinas Mundo Fibra

Sitio web para la empresa **Piscinas Mundo Fibra**, especializada en piscinas de fibra de vidrio.

## Características

- **Cotizador online** — El cliente ingresa dimensiones, color y accesorios.
- **Orden de compra** — Generada al instante con número único, resumen completo y cálculos automáticos de volumen.
- **Contacto WhatsApp** — Botón flotante y envío directo de la cotización por WhatsApp al +56 9 5408 8120.
- **Secciones informativas** — Ventajas, tecnología multicapa y paleta de colores disponibles.

## Stack

- [Next.js](https://nextjs.org/) 16 (App Router)
- TypeScript
- Tailwind CSS
- Lucide React (iconos)
- Desplegado en [Vercel](https://vercel.com/)

## Desarrollo local

```bash
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en el navegador.

## Despliegue en Vercel

1. Conecta el repositorio en [vercel.com](https://vercel.com/)
2. Vercel detecta automáticamente Next.js y configura el build.
3. Cada `push` a `main` despliega automáticamente.
